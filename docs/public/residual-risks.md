# Residual Risks

- Apple 1.0 is narrower than the full PRD matrix.
- The macOS package is unsigned and not notarized; release notes must include
  SHA-256 verification and Gatekeeper expectations.
- Raw QA evidence, host logs, cache files, media renders, and support bundles
  are not included in this public repository.
- Model files are not vendored in this public source repository.
